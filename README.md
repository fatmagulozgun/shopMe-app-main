# EasyShop

Modern bir e-ticaret arayüzü demosu. Proje React, Redux Toolkit ve Tailwind CSS kullanılarak geliştirildi.

## Öne Çıkan Özellikler

- Ürün listeleme, detay sayfası ve kategori filtreleme
- Fiyat sıralama ve anlık ürün arama
- LocalStorage destekli sepet ve favori yönetimi
- Dış API hata verdiğinde yerel yedek veriyle devam eden dayanıklı veri akışı
- Mobil uyumlu, portfolyo sunumuna uygun modern arayüz

## Teknolojiler

- React
- Redux Toolkit
- Tailwind CSS
- React Router
- React Toastify

## Kurulum

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Proje Notu

Bu proje, kullanıcı deneyimi odaklı front-end geliştirme, durum yönetimi ve hata toleranslı veri akışı becerilerini göstermek amacıyla portfolyo için güçlendirilmiştir.
